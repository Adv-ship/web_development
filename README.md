📚 Description

This project demonstrates:

Basic HTML webpage creation
Tables, forms, images, and hyperlinks
CSS styling concepts
XML document creation
XSL/XSLT transformation
Data representation using XML
Displaying XML data in browser using XSL
🛠 Technologies Used
HTML5
CSS3
XML

▶️ How to Run
HTML Files
Download or clone the repository.
Open any .html file in a web browser.
XML with XSL/XSLT
Make sure XML file contains XSL reference:
<?xml-stylesheet type="text/xsl" href="Employees.xsl"?>
Open the XML file directly in browser.
